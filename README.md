A premium, JARVIS-inspired AI agent with a floating holographic core UI.

## Features
- **Holographic Core**: A transparent, always-on-top glowing sphere that reacts to you.
- **Gemini 2.0 Flash Lite**: Advanced reasoning for local system control.
- **System Control**: Open apps, create folders, and search the web via voice/text.
- **Lightweight**: Minimal overhead, high-performance animations.

## Setup
1. **API Key**: Create a `.env` file based on `.env.example` and add your `GOOGLE_AI_API_KEY`.
2. **Install**: Run `npm install`.
3. **Launch**: Run `npm start`.

## Usage
- Click the **Glowing Sphere** to enter a command.
- Command examples:
    - "Open Chrome and search for the latest news in Pakistan."
    - "Create a folder named Project-Echo on my desktop."
    - "Open Spotify."

## Architecture
- **Electron**: Handles the transparent window and system level access.
- **Google Generative AI**: The "Brain" processing your intents.
- **CSS3 Animations**: High-performance "Premium" visual effects.
